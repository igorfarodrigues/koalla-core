# AI Agents Knowledge Base

Este diretório centraliza contexto para agentes de IA contribuírem neste repositório com segurança e consistência.

## Arquivos

- `project-context.yaml`: resumo estruturado do projeto
- `01-projeto-arquitetura.md`: visão geral da arquitetura e fluxos
- `02-boas-praticas.md`: padrões de desenvolvimento
- `03-regras-de-negocio.md`: regras funcionais do domínio
- `04-regras-engenharia-seguranca.md`: regras técnicas e de segurança
- `05-estrutura-de-pastas.md`: mapa de diretórios e responsabilidades
- `06-manual-de-testes.md`: roteiro de testes local e funcional

## Uso recomendado por agentes

1. Ler `project-context.yaml` para contexto rápido.
2. Ler `03-regras-de-negocio.md` e `04-regras-engenharia-seguranca.md` antes de implementar.
3. Validar mudanças seguindo `06-manual-de-testes.md`.
